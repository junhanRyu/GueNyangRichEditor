# MokaEditor
Android Native Rich Editor Project

features :
bold
underline
strikethrough
fontsize
foregroundcolor
backgroundcolor
checkbox
quote
bullet
clickable


Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.junhanRyu:MokaEditor:Tag'
	}
